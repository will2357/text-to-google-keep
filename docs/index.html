text-to-google-keep

One Google Keep note per line — CLI, or a Django + Inertia + React app you run locally. This site on GitHub Pages is static documentation only; the importer always runs on your machine (or your own server).

Install (CLI)

From the repo root:

  • uv sync
  • uv run text-to-google-keep --help
  • Use --oauth for Google OAuth + official Keep API (personal Gmail).

See the repository README for PostgreSQL, Django, Vite, and OAuth setup.

Full web UI

GitHub Pages cannot run Django or PostgreSQL. Run the app locally:

  • npm install && npm run build && uv run python manage.py migrate
  • npm run dev (terminal 1) + uv run python manage.py runserver (terminal 2)
